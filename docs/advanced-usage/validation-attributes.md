These are all the validation attributes currently available in laravel-data.

## Accepted

[Docs](https://laravel.com/docs/9.x/validation#rule-accepted)

```php
#[Accepted]
public bool $closure; 
```

## AcceptedIf

[Docs](https://laravel.com/docs/9.x/validation#rule-accepted-if)

```php

```
